Format FEC
Le FEC (Fichier des Écritures Comptables) est le format obligatoire pour les contrôles fiscaux en France. Il contient toutes vos écritures comptables au format normalisé.
Début de l'exercice (par défaut : 1er janvier)

Informations
Format du fichier

Le fichier généré sera au format texte (.txt) avec séparateur pipe (|).

Nom du fichier

Format : SIRETFECYYYYMMDDYYYYMMDD.txt
Exemple : 12345678900012FEC2025010120251231.txt

Contenu

Le fichier contient vos factures de vente converties en écritures comptables selon le plan comptable français.

  • Envoi à votre expert-comptable
